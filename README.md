# 🧬 UBCB-Core: Registry & Event Management System

**UBCB-Core** is the central web service and database system designed for **Udvaboni Biggan Club, Bahubal**. It serves as a dynamic portal to manage club registries, track active executive members, and organize event participations seamlessly.

## 👥 Executive Members & Roles Covered

The core registry dynamically handles and showcases the leadership profiles of **Udvaboni Biggan Club, Bahubal**, including:
*   **President / General Secretary / Club Secretary**
*   **Robotics Research & Development Secretary**
*   **IT & Research, Innovation Affairs, and Cultural Secretaries**
*   **Active Executive Members & Coordinators**

---

## 🚀 Key Features

*   **Dynamic Registry:** Displays all registered club members with their designation and core operational department in real-time.
*   **Event Tracking:** A dedicated event tracking system showing which member is assigned to or participating in specific science fests, exhibitions, or tech competitions.
*   **Secure Database:** Backed by an optimized database configuration (`club.db`) to ensure fast queries and safe data storage.
*   **Modern Dark UI:** A fully responsive, clean, and user-friendly dark-themed dashboard.

---

## 🛠️ Tech Stack & Architecture

*   **Backend:** Python (Flask/Gunicorn)
*   **Frontend:** HTML5, CSS3 (Templates)
*   **Database:** SQLite (`club.db`)
*   **Deployment:** Render (Web Service) / GitHub Pages
